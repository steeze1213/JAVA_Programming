# Java Programming

학교 수업 자바 프로그래밍을 한 학기 동안 수강하며 실습, 과제로 작성한 코드 모음입니다.  
자바 기초 문법부터 객체지향, GUI, 스레드, 네트워크, DB 연동까지 단계적으로 학습한 개인 학습 기록용 레포지토리입니다.

---

## 사용 교재

명품 JAVA Programming
- 출판 연도: 2018

## 학습 내용 요약

### CHAPTER 01 – 자바 개요 및 개발 환경
- 자바 개념, WORA, 개발 도구, 이클립스 사용법

### CHAPTER 02 – 자바 기본 문법
- 프로그램 구조, 데이터 타입, 연산, 조건문

### CHAPTER 03 – 반복문, 배열, 예외 처리
- 반복문, 배열, 다차원 배열, 예외 처리

### CHAPTER 04 – 클래스와 객체
- 클래스 정의, 생성자, 객체 배열, 접근 지정자, static/final

### CHAPTER 05 – 상속과 다형성
- 상속, 오버라이딩, 추상 클래스, 인터페이스

### CHAPTER 06 – 패키지, 모듈, 기본 클래스
- 패키지, 모듈, Object, Wrapper, String, Math, Calendar

### CHAPTER 07 – 제네릭과 컬렉션
- 제네릭 개념, 컬렉션 프레임워크

### CHAPTER 08 – 입출력 스트림과 파일 I/O
- 문자/바이트 스트림, 버퍼 입출력, 파일 처리

### CHAPTER 09 – 자바 GUI 기초 (Swing)
- 스윙 컴포넌트, 컨테이너, 레이아웃

### CHAPTER 10 – 이벤트 처리
- 이벤트 모델, 리스너, 키/마우스 이벤트

### CHAPTER 11 – 기본 스윙 컴포넌트
- 버튼, 체크박스, 텍스트필드, 리스트, 콤보박스 등

### CHAPTER 12 – 그래픽
- Graphics, 도형/이미지 그리기, 페인팅 메커니즘

### CHAPTER 13 – 스레드와 멀티태스킹
- 멀티스레딩, 동기화, 스레드 생명주기

### CHAPTER 14 – 고급 스윙 컴포넌트
- 메뉴, 다이얼로그, 탭, 오디오 처리

### CHAPTER 15 – 네트워크 프로그래밍
- TCP/IP, 소켓, 서버-클라이언트 통신

### CHAPTER 16 – JDBC 프로그래밍
- 데이터베이스 개념, MySQL, JDBC 연동
